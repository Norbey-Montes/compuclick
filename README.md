**Nombre del aprendiz:** Norbey Steep Montes Ramírez

## Descripción general
CompuClick es un sistema de gestión diseñado para el control de inventarios, ventas y proveedores. El proyecto implementa roles diferenciados para Administrador, Usuario y Proveedor, asegurando una gestión eficiente y organizada de los recursos.

## Estructura utilizada
El proyecto se organiza de la siguiente manera para separar la lógica de la interfaz:
- `public/`: Contiene la interfaz pública y el archivo principal index.php.
- `app/`: Contiene la lógica de negocio, dividida en:
    - `controllers/`: Gestión de controladores.
    - `models/`: Gestión de modelos de datos.
    - `views/`: Gestión de las vistas.
- `docs/`: Contiene la documentación técnica y archivos de soporte del proyecto.
